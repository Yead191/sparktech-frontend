# SparkTech Agency Frontend

Welcome to the **SparkTech Agency Frontend** project! This is the official web application for SparkTech Agency, designed to provide an engaging, fast, and modern user experience. Built with the latest technologies like Next.js 15, React 19, and Tailwind CSS v4, it ensures high performance, seamless animations, and a responsive design.

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **UI Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/), [Ant Design](https://ant.design/)
- **Animations & Scrolling**: [GSAP](https://gsap.com/), [Lenis](https://lenis.studiofreight.com/)
- **Carousels/Sliders**: [Swiper](https://swiperjs.com/)
- **Icons**: [Lucide React](https://lucide.dev/), [React Icons](https://react-icons.github.io/react-icons/)
- **Toast Notifications**: [Sonner](https://sonner.emilkowal.ski/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🌟 Key Features

- **Home**: A dynamic landing page showcasing the agency's core offerings.
- **Services**: Detailed breakdown of the specialized services provided by SparkTech.
- **Company**: Information about the company, mission, and vision.
- **Career**: Open job positions and career categories.
- **Life at Spark**: A glimpse into the company culture and work environment.
- **Contact**: Interactive contact forms and location details.
- **Smooth Animations**: High-quality scroll animations powered by GSAP and Lenis.

## 📂 Project Structure

```text
src/
├── app/          # Next.js App Router layout and pages
```

## 🛠️ Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (version 20+) and `npm`, `yarn`, `pnpm`, or `bun` installed on your machine.

### Installation

1. Clone the repository and navigate to the project directory:
   ```bash
   cd sparktechagency-web-front-end
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

### Running the Development Server

Start the development server with Turbopack for faster reloads:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [https://www.sparktech.agency/](https://www.sparktech.agency/) with your browser to see the application in action.

## 📜 Scripts

- `npm run dev`: Starts the development server using Turbopack.
- `npm run build`: Creates an optimized production build.
- `npm start`: Starts the application in production mode.
