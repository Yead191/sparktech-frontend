# Sparktech Agency

🔗 **Live Website:** [sparktech.agency](https://www.sparktech.agency/)

Developed and engineered the frontend architecture for **Sparktech Agency** — a modern digital agency platform focused on innovation, strategy, and impactful user experiences. The application was built with a strong emphasis on performance, scalability, responsive design, and clean UI implementation, transforming creative concepts into a seamless and engaging digital experience.

From crafting reusable components to optimizing user interactions, this project reflects a balance between modern design aesthetics and efficient development practices. The frontend is designed to deliver smooth navigation, interactive experiences, and a visually polished interface that aligns with Sparktech Agency’s forward-thinking vision.

## 🚀 Tech Stack & Tools

This project is built with the latest modern web technologies to ensure a high-performance, engaging, and scalable user experience:

### Core Frameworks & Languages

- **[Next.js 15](https://nextjs.org/)**: React framework with App Router and Turbopack for lightning-fast development.
- **[React 19](https://react.dev/)**: The latest version of the popular UI library.
- **[TypeScript](https://www.typescriptlang.org/)**: Strongly typed programming language that builds on JavaScript.

### Styling & UI Libraries

- **[Tailwind CSS v4](https://tailwindcss.com/)**: Utility-first CSS framework for rapid UI development.
- **[Ant Design](https://ant.design/)**: Enterprise-level UI design language and React component library.
- **[Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)**: Comprehensive icon libraries for beautiful SVG icons.

### Animations, Interactions & Scrolling

- **[GSAP](https://gsap.com/) (GreenSock Animation Platform)**: Professional-grade animation library for complex UI animations.
- **[Lenis](https://lenis.studiofreight.com/)**: A lightweight, robust smooth scrolling library.
- **[@use-gesture/react](https://use-gesture.netlify.app/)**: Hook for binding rich mouse and touch gestures.

### Carousels & Data Display

- **[Swiper](https://swiperjs.com/)**: The most modern mobile touch slider.
- **[React Fast Marquee](https://www.npmjs.com/package/react-fast-marquee)**: Lightweight React component that utilizes CSS animations to create silky smooth marquees.

### Utility & Feedback

- **[Sonner](https://sonner.emilkowal.ski/)**: An opinionated toast component for React for beautiful notifications.

## 🛠️ Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (version 20+) and your preferred package manager installed (`npm`, `yarn`, `pnpm`, or `bun`).

### Installation

1. Clone the repository and navigate to the project directory:

   ```bash
   cd sparktechagency-web-front-end
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

Start the development server with Turbopack for faster reloads:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application in action.

---

## 📜 Available Scripts

- `npm run dev`: Starts the development server using Turbopack.
- `npm run build`: Creates an optimized production build.
- `npm start`: Starts the application in production mode.
